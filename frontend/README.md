# Frontend do Smart Working System

Este é o projeto de frontend para o Smart Working System, construído com React e Vite.

## Como Executar

1. **Navegue até a pasta `frontend`:**
   ```bash
   cd frontend
   ```

2. **Instale as dependências (se ainda não o fez):**
   ```bash
   npm install
   ```

3. **Execute o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

O aplicativo estará disponível em `http://localhost:5173`.
